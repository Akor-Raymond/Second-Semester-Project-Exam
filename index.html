Welcome to Akor Raymond's Landing Page

Following the completion of my second-semester cloud engineering coursework and Learning Modules (LMs) at Altschool, I was assigned a project to design and develop a prototype for a web application as if I were a member of a startup team.

Project Description

Provisioning a Linux Server with a simple HTML page.

The Tasks

Provisioning the server

  • Using an AWS cloud platform, set up a Linux server
  • Install a Linux distribution (Ubuntu)

Setting up the web server

  • Install a web server (Nginx)

HTML page deployment

  • Create a simple HTML page with the following informations
    • My name
    • The project title
    • A brief description of the project
    • My full bio with every interesting information about me
  • Deploy the HTML page on the server

Networking

  • Configure the server to allow HTTP traffic (port 80)
  • Providing the public IP address so that the page can be accessed from any browser
